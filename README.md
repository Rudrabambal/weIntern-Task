# WeIntern - Multi-Utility Workspace
file:///C:/Users/rudra/calculator/index.html 
A premium, modern, glassmorphic single-page web application featuring three essential utilities: a **Precision Calculator**, a **Focus To-Do List**, and an **Interactive Counter**.

---

## 🚀 Key Features

* **Glassmorphic UI**: Beautiful dark-themed aesthetic with floating glowing spheres, smooth animations, and transitions.
* **Precision Calculator**: 
  * Supports standard arithmetic operations (+, -, *, /, %).
  * Dynamic equations history log.
  * Full keyboard support with visual key press feedback.
* **Focus To-Do List**:
  * Add, complete, and delete tasks (with slide-out animations).
  * Filter tasks by category: All, Active, and Completed.
  * Animated progress bar tracking task completion.
  * Persistent storage using browser `localStorage`.
* **Interactive Counter**:
  * Bouncing tactile count displays.
  * Adjust custom step sizes and presets (+1, +5, +10, +50).
  * Real-time activity logging in the sidebar.
